# Manny Toolbox

Quick Start: https://mannylee.github.io/manny-toolbox/

An interesting collection of web utilities designed to supercharge your productivity! Transform messy data, decode complex URLs, and beautify JSON with lightning-fast tools that handle the tedious work so you can focus on what matters.

## Tools

### [List Magic](https://mannylee.github.io/manny-toolbox/?page=listmagic)
![List Magic screenshot](https://mannylee.github.io/manny-toolbox/img/screenshots/listmagic.jpg)

Converts a line of text (based on a delimiter character) into a list of individual text items, and back.

### [URL Decoder](https://mannylee.github.io/manny-toolbox/?page=urldecoder)

Decodes URL-encoded strings and extracts URL search parameters. Features automatic parsing of nested JSON values within URL parameters.

### [JSON Unfurler](https://mannylee.github.io/manny-toolbox/?page=jsonunfurler)

Parses and formats JSON with proper indentation. Automatically detects and unfurls nested JSON strings within the main JSON object, converting escaped JSON into readable nested structures.

### [AWS IP Checker](https://mannylee.github.io/manny-toolbox/?page=awsipchecker)

Check if an IP address belongs to AWS infrastructure. Supports both IPv4 and IPv6 addresses and provides detailed information about AWS regions and services.

### [Kiro-CLI Formatter](https://mannylee.github.io/manny-toolbox/?page=kirocliformatter)

Reformats messy Kiro-CLI output into clean, readable prose. Strips leading whitespace from every line, flattens hard line breaks into single spaces, converts runs of two consecutive spaces into paragraph breaks, and collapses any leftover indentation that follows a newline.
